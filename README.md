# PageOff — Flixchart for Books

“Which book do you prefer?”  
PageOff lets users compare books head-to-head and builds rankings with a simple Elo-style system.

> **Status:** Phase 0 (repo + local env). MVP in active development.

---

## Tech Stack (MVP)
- **Backend:** FastAPI (Python)
- **Frontend:** Jinja2 templates + HTMX (no React)
- **Database:** SQLite (SQLAlchemy)
- **Dev server:** Uvicorn
